# barkRelay

音声入力を文字起こしして、AI agentに渡すシステム

## 概要

BarkRelayは音声入力を受け取り、それを文字に変換してAI agentに送信するための中継システムです。

## 機能

- [ ] 音声入力キャプチャ
- [ ] リアルタイム音声認識・文字起こし
- [ ] AI agentとの連携インターフェース

## 開発方針

- 全ての機能はGitHub issueとして管理
- 機能実装時は専用ブランチを作成してPull Requestで進行

## 技術スタック

- 音声処理: 検討中
- 音声認識: 検討中
- AI連携: 検討中
